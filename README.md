# SteepWood Webapp

Next.js site for [steepwood.com.au](https://steepwood.com.au).

## Local development

From the repo root:

```bash
cd webapp
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to preview changes before deploying.

You can also run from the repo root:

```bash
npm run dev
```

## Deploy

Pushes to `main` on GitHub deploy automatically via Vercel. The Vercel project root directory is set to `webapp`.
